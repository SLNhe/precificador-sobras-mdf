# Precificador de Sobras de MDF

Aplicação web desenvolvida com **HTML, CSS e JavaScript** para calcular o valor proporcional de sobras de chapas de MDF de forma rápida e prática.

O objetivo do projeto é facilitar a precificação de retalhos de MDF, permitindo informar as dimensões da sobra e o valor da chapa inteira para obter um preço proporcional. Após o cálculo, o orçamento pode ser **copiado para a área de transferência** ou **compartilhado diretamente pelo WhatsApp**, agilizando o atendimento ao cliente.
